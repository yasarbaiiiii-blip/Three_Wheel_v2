# DXF_Three_Wheel

## Rover Three Wheel Native

This repository now runs as an Expo React Native app styled with NativeWind.

## Install

```bash
npm install
```

## Run

```bash
npm run start
```

For device targets:

```bash
npm run android
npm run ios
```

## Structure

- `App.tsx`: main app shell
- `src/components`: native UI components
- `src/theme/colors.ts`: shared palette

## Note

Some old web files may still appear in the tree if Windows is holding file locks on them, but the app entrypoint and runtime are now fully React Native.
